Введение в JavaScript

Определение

JavaScript (джаваскрипт) — прототипно-ориентированный сценарный язык программирования.

JavaScript обычно используется как встраиваемый язык для программного доступа к объектам приложений. Наиболее широкое применение находит в браузерах как язык сценариев для придания интерактивности веб-страницам.

Основные архитектурные черты: динамическая типизация, слабая типизация, автоматическое управление памятью, прототипное программирование, функции как объекты первого класса.

Если вы не знакомы с программированием, из определения достаточно сложно понять хоть что-то, и это нормально. По мере дальнейшего изучения JavaScript мы будем закрывать непонятные конструкции из определения, но изначально важно знать следующее: программы на JavaScript представляют собой обычный текст и выполняются интерпретатором, который может быть встроен в браузер или любые другие программы.

Краткая история

JavaScript был создан в 1995 Бренданом Эйхом внутри компании Netscape c целью обеспечить «язык для склеивания» составляющих частей веб-ресурса: изображений, плагинов, Java-апплетов, который был бы удобен для веб-дизайнеров и программистов, не обладающих высокой квалификацией.

Вследствие рождения в браузере, одной из основных задач JavaScript была манупуляция с HTML и CSS, что добавило на веб-страницы интерактивность.

Следующим важным шагом для JavaScript стало появление возможности обращаться к серверу без перезагрузки страницы. Данная технология называлась AJAX. После её появления на JavaScript начали писать не только логику небольших изменений HTML или CSS кода страницы, но и некоторую сложную логику поведения страницы, что в совокупности с постоянно растущей скоростью работы JavaScript позволило писать такие сложные веб-приложения, как Gmail.

В 2009 году вышла программная платформа Node.js, которая дала возможность легко использовать JavaScript для написания полноценных программ вне браузера, после чего стало можно писать веб сервисы полностью на JavaScript.
Также Node.js способствовала появлению огромного количества консольных программ на JavaScript.

На данный момент JavaScript является одним из самых популярных языков программирования в мире.
